## Features

## Infrastructure

* Build / Hosting: [Vercel](https://vercel.com/)
* Headless CMS: [microCMS](https://microcms.io/)
* Push Notification: [OneSignal](https://onesignal.com/)

## Requires

* Node.js >= v14
* npm v6
